## nf-core/modules MaxBin2 tests

- Module page: https://nf-co.re/modules/MaxBin2
- Tests: https://github.com/nf-core/modules/tree/master/modules/nf-core/maxbin2/tests

characterization tests, since nf-core actively runs them in CI.
